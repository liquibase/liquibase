# Three-Strike Rule

## When to Stop and Ask for Help

If you've tried 3 different approaches without success:
1. **STOP** attempting
2. **DOCUMENT** what you've tried
3. **ASK** for help

## Signs You Need Help
- 🔄 Making the same error repeatedly
- 🤔 Guessing about implementation details
- 📚 Documentation is unclear or missing
- 🔍 Can't find a working pattern
- ⏰ Spending >30 min on a "simple" task
- 🎯 Assumptions keep proving wrong

## Why This Matters
- **No Guessing**: When uncertain, validate assumptions before proceeding
- **Early Help Requests**: Better to ask early than waste time on wrong approaches
- **Complete Solutions**: Time spent on shortcuts usually doubles total time

## Validated Through
Multiple instances of wasted time during Snowflake extension development when this rule wasn't followed.
