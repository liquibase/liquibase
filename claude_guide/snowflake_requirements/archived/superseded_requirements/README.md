## Archive Policy
Files are moved here when:
1. They are planning documents superseded by completed implementations
2. They contain duplicate information available in production-ready files
3. They are implementation status reports rather than requirements templates
4. They are no longer needed for active development

Files in this archive are kept for historical reference but should not be used for active development.
